## 1. Agent / AI 框架层

| 组件 | 选型 | 版本 | 说明 |
|------|------|------|------|
| Agent 框架 | Dify | 0.8+ (社区版) | 可视化 Agent 编排，支持 LLM 切换、知识库 RAG、工具调用 |
| 备选框架 | LangChain | 0.2+ | 复杂场景下自定义 Agent 流程 |
| LLM API | 通义千问 (Qwen-Max/Qwen-Plus) | — | 中文能力强、成本低 |
| 备用 LLM | DeepSeek-V2 | API | 备选模型，按用量切换 |
| 向量数据库 | Milvus / pgvector | Milvus 2.4+ | RAG 知识库检索 |
| 文本嵌入 | BGE-m3 / text-embedding-3 | — | 中文向量检索优化 |

## 2. 企业微信对接层

| 组件 | 选型 | 说明 |
|------|------|------|
| 企微 SDK | WeCom SDK (Python: wechatwork) | 会话存档、群机器人、应用消息 |
| 消息回调 | Flask Webhook | 接收企微事件回调（消息/成员变更） |
| 会话存档 API | 企微官方 REST API | 拉取聊天记录（需购买会话存档） |
| 通讯录 API | 企微官方 REST API | 同步部门/成员信息 |
| 群管理 API | 企微官方 REST API | 群机器人管理、群消息推送 |

## 5. 私域运营核心模块

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| SOP 引擎 | Celery Beat 定时任务 + 规则引擎 | 自动化运营流程调度 |
| 用户标签系统 | PostgreSQL JSONB + 规则匹配 | 动态标签 + 行为标签 |
| 意图识别 | Dify Agent + Qwen LLM | 自然语言意图分类 |
| 智能回复 | Dify Workflow + RAG | 知识库检索 + 模板回复 |
| 精准触达 | Celery 任务队列 + 企微 API | 按标签/条件群发 |

## 6. 数据看板

| 指标 | 计算方式 |
|------|----------|
| 会话量 | 每日/每周/每月会话统计 |
| 解决率 | AI 自动回复中用户未追问的比例 |
| 转人工率 | AI 无法处理转人工客服的比例 |
| 用户活跃度 | DAU/MAU、会话时长分布 |
| 标签覆盖率 | 被打标签用户 / 总用户 |
| SOP 执行率 | SOP 触发成功率、完成率 |
