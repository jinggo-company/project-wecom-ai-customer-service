## 需求概述

源自猪八戒网真实需求：企微自动客服项目（预算 ¥8,400）和自助洗车机企微引流项目。核心场景是帮助企业通过企业微信实现自动客服响应和私域引流，降低人工客服成本，提升客户转化效率。

2026 年中国中小企业数字化转型渗透率突破 65%，企业微信用户规模持续增长，成为私域运营核心渠道。传统客服人力成本持续攀升（一线城市月薪 8,500+），AI 大模型使自动客服从"关键词匹配"升级为"语义理解+多轮对话"。

**立项理由**：需求真实明确（多个猪八戒网需求验证），技术可行性高（企微 API 开放程度高，Dify/Coze 等成熟方案可用），商业价值大（可产品化定价，市场空间超 50 亿元）。已有 AI 部署服务能力（P-2026-00007）和 n8n 自动化底座（P-2026-00001）可复用。

## 技术栈

- **Agent 框架**: Dify 0.8+（社区版，可视化编排）/ LangChain 0.2+（复杂场景）
- **LLM**: 通义千问（Qwen-Max/Qwen-Plus）为主，DeepSeek-V2 备用
- **知识库 RAG**: Milvus 2.4+ / pgvector + BGE-m3 文本嵌入
- **后端**: Python 3.11+ / FastAPI / Celery + Redis / PostgreSQL 16
- **前端**: React 18 + Ant Design 5 + ECharts 5
- **企微对接**: 企微官方 REST API + WeCom SDK（Python: wechatwork）
- **部署**: Docker / Docker Compose
- **CI/CD**: GitHub Actions

### 关键决策
1. **优先 Dify**：低代码降低交付门槛，适合快速搭建 MVP；复杂场景用 LangChain 补充
2. **通义千问为主**：中文能力强、API 稳定、成本低于 GPT-4
3. **RAG 优于微调**：企业知识库更新频繁，RAG 方案维护成本低、响应快
4. **规则+AI 混合**：规则式回复兜底保证稳定性，AI 处理长尾复杂问题

## 验收标准（Acceptance Criteria）

### AC-1：企微消息接入
- 成功接收企微回调消息（文本/图片/链接），延迟 ≤ 2 秒
- 支持消息去重（企微重试机制），不重复处理同一条消息
- 支持 ≥ 5 个企微应用同时接入

### AC-2：AI 自动回复
- 接入通义千问/DeepSeek，支持语义理解回复
- 回复生成时间 ≤ 5 秒（P95），含 LLM 调用 + 后处理
- 支持多轮对话上下文保持（≥ 5 轮）
- 回复内容经过安全过滤（无敏感词、无违规内容）

### AC-3：知识库 RAG
- 支持 PDF/Word/Excel/TXT 文档上传，自动提取文本并入库
- 向量检索准确率 ≥ 85%（人工标注测试集验证）
- 知识库更新后 ≤ 5 分钟内生效（增量向量化）
- 支持 ≥ 10 万条知识库条目

### AC-4：规则式自动回复
- 支持关键词匹配（精确/模糊/正则表达式）
- 规则优先级可配置，支持 AND/OR 组合条件
- 规则引擎命中率 ≥ 95%（对预设规则集的测试）

### AC-5：混合回复策略
- 规则匹配优先，未匹配则走 AI 回复，AI 失败则走兜底回复
- 回复切换逻辑正确，无重复回复/无回复
- 支持配置 AI 回复置信度阈值（低于阈值走规则/人工）

### AC-6：会话管理 + 人工接管
- 会话列表实时更新，延迟 ≤ 3 秒
- 支持按时间/客户/状态筛选会话
- 人工接管后 AI 停止回复，人工回复后可一键交还 AI
- 会话记录保留 ≥ 90 天

### AC-7：数据看板
- 实时显示：今日消息量、AI 回复量、人工回复量、转人工率
- 热门问题 TOP 20，按日/周/月统计
- AI 回复准确率统计（用户点赞/点踩反馈）
- 数据刷新频率 ≤ 1 分钟

### AC-8：系统性能与稳定性
- 支持并发 ≥ 100 会话/秒，P95 响应时间 ≤ 8 秒
- 服务可用性 ≥ 99.5%（月度）
- 企微 API 限流自动重试（3 次，指数退避）
- 异常消息不丢失（Dead Letter Queue 机制）

## 风险与应对

| 风险 | 影响 | 概率 | 应对措施 |
|------|------|------|----------|
| **企微 API 限制**：部分功能受平台政策约束，可能随时变更 | 高 | 中 | 抽象消息接入层，支持快速切换渠道；关注企微官方公告 |
| **AI 回复质量**：知识库质量差导致 AI 回复不准确 | 中 | 高 | 提供知识库质量评估工具，引导客户优化数据；设置 AI 置信度阈值 |
| **竞品众多**：市场上已有大量企微 SCRM 产品 | 中 | 高 | 差异化：AI 深度集成 + 低门槛部署 + 价格优势 |
| **LLM 成本控制**：高频调用导致 API 成本上升 | 中 | 中 | 缓存热门问答、降级小模型、按量计费监控 |
| **客户数据安全**：企业客户对话数据敏感 | 高 | 中 | 数据加密存储、访问权限控制、合规审查、数据不外传 |
